小根堆，根<=左、右

大根堆，根>=左、右

堆存储在数组中，i为下标从0开始：

  • i >= int ( n / 2 )，则 i 结点时叶结点

  • i=0，结点 i 是根节点；否则结点(int) [ ( i - 1 ) / 2 ]是 i 的父节点；

  • 若2 * i + 1 > n - 1，则 i 没有左孩子

  • 若2 * i + 2 < n - 1，则 i 没有右孩子
